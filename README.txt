# 🚀 Curso de GitHub - UNIBRA

## 🎓 Sobre o Curso

O **Curso de Git e GitHub (40 horas)** oferecido pela **UNIBRA** tem como objetivo capacitar estudantes da área de tecnologia para utilização prática e eficiente de ferramentas de versionamento de código e colaboração em projetos.

---

## 📝 Conteúdo Programático

1. **Introdução ao GitHub**
   - Conceitos de versionamento
   - Instalação e configuração do GitHub Desktop
   - Diferenças entre Git e GitHub

2. **Comandos Básicos do Git**
   - git init, git clone, git status
   - git add, git commit
   - git push, git pull, git fetch

3. **Branches e Merge**
   - Criação e gerenciamento de branches
   - Merge de branches
   - Resolução de conflitos

4. **GitHub na Prática**
   - Criação de repositórios remotos
   - Pull Request e colaboração

5. **Boas Práticas**
   - Mensagens de commit
   - .gitignore e licenciamento
   - Proteção de branch e fluxo de trabalho

6. **Integrações**
   - Integração com VS Code

7. **Projeto Final**
   - Desenvolvimento de projeto colaborativo com GitHub

---

## 🎯 Objetivos de Aprendizagem

✅ Compreender os conceitos fundamentais de Git  
✅ Utilizar comandos do GitHub 
✅ Criar, gerenciar e colaborar em repositórios GitHub  
✅ Aplicar boas práticas de versionamento em projetos reais

---

## 🏅 Certificação

✔️ Certificado de **40 horas** emitido pela **UNIBRA** mediante participação e realização das atividades propostas.
